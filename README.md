## 🚀 What This Is

This is a **plug-and-play website system** designed to turn a simple intake form into a beautiful, high-converting one-page site.

No fluff. No bloated builders. Just:
- clean design
- clear messaging
- strong calls to action

---

## 🧠 How It Works

1. **Client fills out intake form** (`intake.html`)
2. You collect:
   - business info
   - services
   - branding
   - photos
3. You generate a **conversion-optimized site** using that data
4. Deploy instantly via Netlify

## 🛠️ Deploy in < 2 Minutes

### Option 1: Netlify (Recommended)

1. Go to https://netlify.com
2. Click **“Add new site” → Deploy manually**
3. Drag & drop your project folder
4. Done ✅

Your forms will automatically work.

---

### Option 2: GitHub + Netlify

1. Push repo to GitHub
2. Connect to Netlify
3. Auto-deploy enabled
